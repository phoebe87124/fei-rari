# Installation
```
npm install
```

# How to Use
## Set Api Key in .env
```
// .env
API_KEY = xxxxxx
```

## Run Script
```
npx hardhat test
```

## Run Test with Gas Report
```
REPORT_GAS=true npx hardhat test
```
